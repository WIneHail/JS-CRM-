
# JS-CRM — Простая CRM-панель на Vanilla JavaScript

Курсовой проект по JavaScript (Skillbox). Учебное приложение для управления клиентами: создание, редактирование, поиск, фильтрация.

## Технологии
- Vanilla JavaScript (ES6+)
- HTML5 + SCSS (BEM)
- LocalStorage (персистентность данных)
- MVC-подобная архитектура

## Возможности
- Добавление/редактирование/удаление клиентов
- Поиск по имени, email, телефону
- Валидация полей формы (на лету и при отправке)
- Сохранение данных между сессиями
